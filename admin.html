Getting Started

1
Upload Data
Go to the Training section in the sidebar and upload a corpus file (TXT format).
2
Configure Runtime
Visit the Controls panel to enable or disable runtime features like token logging or heuristic assists.
3
Start Training
Click "Restart & Retrain" to begin the training process. Monitor progress via the Log panel.
4
Interact
Use the Chat panel to test your model with prompts and get real-time responses.
5
Save Progress
Export snapshots regularly to backup your model state and restore it later.
